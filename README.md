# Acadex — Academic Publication Management System

A full-stack Academic Publication Management System built with PHP, MySQL, and HTML/CSS. Features a Danzan-inspired horizontal scroll landing page, research paper management, author tracking, citation network, and analytics dashboard. Built as a DBMS college project.

## Tech Stack
- PHP
- MySQL
- HTML / CSS / JavaScript
- XAMPP (local server)

## Features
- 🎬 Cinematic horizontal scroll landing page
- 📊 Research dashboard with live stats
- 📄 Paper management with fulltext search
- ✍️ Author tracking with h-index
- 🔗 Citation network with self-citation prevention
- 📈 Analytics — venue stats, impact factors

## Database Concepts
Normalization, Foreign Keys, Triggers, Views, Stored Procedures, Transactions, FULLTEXT Search, Aggregate Queries, Joins
